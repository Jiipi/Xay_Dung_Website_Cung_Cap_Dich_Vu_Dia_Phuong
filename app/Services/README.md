# Services

Dat cac service theo tung domain nghiep vu.

Nguyen tac:
- Controller chi nhan request, authorize, validate, goi service.
- Service chua use case nghiep vu va phoi hop nhieu model/repository.
- Khong dua query phuc tap truc tiep vao controller.

Goi y dat ten:
- `Category/CategoryService.php`
- `Service/ServiceCatalogService.php`
- `Booking/BookingService.php`
- `Provider/ProviderOnboardingService.php`
- `Recommendation/RecommendationService.php`

Thu muc `Shared` dung cho logic duoc tai su dung giua nhieu domain.
